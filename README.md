# Bharat Votes 🇮🇳

**Bharat Votes** is a premium, interactive web application designed to educate and guide Indian citizens through the democratic process. It provides a comprehensive, multi-language experience covering everything from voter registration to election results.

![Bharat Votes Preview](assets/parliament_hero.png)

## ✨ Features

- **🏛️ Democratic Structure**: Visual hierarchy of Indian governance from Local Panchayats to the National Parliament.
- **📅 Interactive Timeline**: Step-by-step guide through the election phases: Announcement, Nominations, Campaigning, Polling, Counting, and Government Formation.
- **🌍 Multi-Language Support**: Fully localized in **English**, **Hindi (हिंदी)**, and **Marathi (मराठी)**.
- **🤖 Bharat AI Assistant**: A dedicated election expert powered by the **Gemini API** to answer any user queries in real-time.
- **📍 Regional Hub**: Get tailored election news and voting instructions based on your specific State, District, and Taluka.
- **🚶 Voter Journey**: A guided walk-through for new and existing voters (Registration, Preparation, Election Day, and Beyond).
- **✅ Eligibility Checker**: A quick tool to check if you meet the requirements to vote.
- **📰 Live News**: Stay updated with the latest election news and regional updates.
- **❓ Comprehensive FAQ**: Answers to common questions about NOTA, EVMs, ID requirements, and more.
- **🎨 Premium Design**: Modern UI with glassmorphism, smooth animations, and a dynamic patriotic flag background.

## 🚀 Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Edge, Safari).
- (Optional) A **Gemini API Key** to enable the AI Assistant. You can get one for free at [Google AI Studio](https://aistudio.google.com/app/apikey).

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/election.git
   ```
2. Open the project folder:
   ```bash
   cd election
   ```
3. Open `index.html` in your browser.

## 🛠️ Technology Stack

- **Frontend**: HTML5, CSS3 (Vanilla), JavaScript (ES6+).
- **Styling**: Modern CSS with CSS Variables, Flexbox, Grid, and Glassmorphism.
- **AI Integration**: Google Gemini API.
- **Fonts**: Inter, Outfit, and Roboto (via Google Fonts).
- **Icons/Graphics**: Custom SVG icons and premium assets.

## 📁 Project Structure

## 🔒 Privacy & Security

- **Local Storage**: Your Gemini API key is stored only in your browser's local storage and is never sent to any server other than Google's API.
- **No Tracking**: We do not collect any personal data or track your interactions.
